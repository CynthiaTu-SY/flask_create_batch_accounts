Welcome!

Introduction

This website allow user to create a batch of user accounts by uploading a user. After the accounts are created, the account owner can log in to check his/her groupmates. This project is trying to demostrating how a tester can create tons of account in short time during UAT to test the access control of a tested system.

Instruction for use

The process is as follows:

1. Upload a TXT file with a list of username:password:groupname.

a. single file upload

b. only allow .txt file

c. empty file is not accepted

d. user list example

user1:pd1:gp1

user2:pd2:gp1

user2:pd2:gp2

user3:pd3:gp2

2. After accounts created, you can go to login page to check groupmates of different users.

Case Handling

1. If ueser already exist in DB,

a. link user to the new group or existing group if the password match the DB record

b. ignore if the password DO NOT match the DB record

2. Create account in DB for new user and link the user to the new group or existing group
